### Java statement

#### Loops

while loop

```
		// java loop
		int count = 0;
		while (count < 5) {
			System.out.println(count);
			count ++;
		}
```

for loop
```
		int[] myList = {2,3,5,8};
		myList[1] = 10;
		for (int i: myList) {
			System.out.println(i);
		}
	}
```
